Recupere une cle d'un fichier de cles sous la forme d'un chemin de fichier complet. La clé peut soit être un fichier relatif au thème courant, soit un chemin començant par '~', soit un chemin complet, soit vide auquel cas le chemin d'un fichier par defaut est renvoye s'il est specifie.

Paramètres:
pKeyFile le fichier de cles.
cGroupName le com du groupe.
cKeyName le nom de la cle.
bFlushConfFileNeeded est mis a TRUE si la cle est manquante.
cDefaultGroupName nom de groupe alternatif, ou NULL si aucun autre.
cDefaultKeyName nom de cle alternative, ou NULL si aucune autre.
cDefaultDir si la cle est vide, on prendra un fichier par defaut situe dans ce repertoire. (optionnel)
cDefaultFileName si la cle est vide, on prendra ce fichier par defaut dans le repertoire defini ci-dessus. (optionnel)
Renvoie:
le chemin complet du fichier, a liberer avec g_free().
