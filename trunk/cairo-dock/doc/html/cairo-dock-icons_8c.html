void cairo_dock_free_icon (  Icon *  icon  ) 
 

Libere toutes les ressources liees a une icone, ainsi que cette derniere. Le sous-dock pointee par elle n'est pas dereferencee, cela doit etre fait au prealable.

Paramètres:
icon l'icone a liberer.

int cairo_dock_compare_icons_order (  Icon *  icon1,
Icon *  icon2
) 
 

Compare 2 icones grace a la relation d'ordre sur le couple (position du type , ordre).

Paramètres:
icon1 une icone.
icon2 une autre icone.
Renvoie:
-1 si icone1 < icone2, 1 si icone1 > icone2, 0 si icone1 = icone2 (au sens de la relation d'ordre).

GList* cairo_dock_sort_icons_by_order (  GList *  pIconList  ) 
 

Trie une liste en se basant sur la relation d'ordre sur le couple (position du type , ordre).

Paramètres:
pIconList la liste d'icones.
Renvoie:
la liste triee. Les elements sont les memes que ceux de la liste initiale, seul leur ordre a change.

GList* cairo_dock_sort_icons_by_name (  GList *  pIconList  ) 
 

Trie une liste en se basant sur la relation d'ordre alphanumerique sur le nom des icones.

Paramètres:
pIconList la liste d'icones.
Renvoie:
la liste triee. Les elements sont les memes que ceux de la liste initiale, seul leur ordre a change. Les ordres des icones sont mis a jour pour refleter le nouvel ordre global.
